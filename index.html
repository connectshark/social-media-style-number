格式化大數字

使用 new Intl.NumberFormat()

基礎使用方式

new Intl.NumberFormat()或是直接使用Intl.NumberFormat()

new Intl.NumberFormat().format(601900)

new Intl.NumberFormat("en-US", { style: 'percent', }).format(3500)

new Intl.NumberFormat('zh-TW', { notation: 'compact' }).format(7384927193)

new Intl.NumberFormat('en', { maximumFractionDigits: 1, }).format(4.67567)

new Intl.NumberFormat('zh-TW', { notation: 'compact', style: 'currency', currency: 'TWD' }).format(846284765)
